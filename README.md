# move-ahead-with-claude

Daily security challenges auto-generated by a scheduled Claude agent every night at 10pm.

## Topics
- **DevSecOps** — policy-as-code, supply chain security, IaC hardening, threat modeling CI/CD
- **Cloud Security** — IAM escalation, CSPM, cross-account attacks, cloud forensics
- **Kubernetes Security** — RBAC, runtime detection, network policies, container escapes
- **Detection Engineering** — Sigma/YARA rules, detection-as-code, threat hunting, MITRE ATT&CK

## How It Works
A scheduled Claude agent runs nightly, generates a challenge scoped to 1–1.5 hours, and pushes it to `challenges/YYYY-MM-DD.md`. Track your progress in `progress.md`.

## Structure
```
challenges/          ← daily challenge files
progress.md          ← mark challenges done, add notes
CLAUDE.md            ← agent instructions
```
